    Borders, Containers & Rules

    variables

    border

    $border: 1px solid $grey-light !default;

    Description

    Borders should be used sparingly and be kept thin, whenever possible. A width of 1px, solid, and grey ($grey-light) is the preferred choice.

    css

    .apply-border

    .apply-border { ... }

    Description

    Applies borders, padding, and margin

    Example

    Note that apply-border does not style corners. Rounded corners are only used for regional alerts

    <div class="apply-border">
  This is an example of proper border color and weight when applied to a container.
</div>

    Colors

    Color is used to signal structure on a page, to highlight or emphasize important information or elements, and to display different states of an interface.

    Use color intentionally. It's not decoration. Color is supposed to be functional. For most of our colors, the following points should apply:

    • Every color has specific guidelines for meaning. We honor these guidelines in order to maintain consistency across the sites.
    • Accessibility is paramount when combining colors. Text should be readable and meet WCAG AA or AAA guidelines.
    • Do not add color if it is not needed. Adding too much color is distracting and can lead to cognitive confulsion. Where possible, let the content do the work.
    • Color should not convey unique and critical information. WCAG color guidelines

    actionable-colors

    $actionable-colors: (
  'default-button': $blue-alt,
  'defalut-subtle-button': $blue-alt-lightest,
  'alertnate-button': $brand-orange,
  'default-link': $blue-alt-dark,
  'link-on-dark-background': $blue-alt-lightest,
) !default;

    Description

    Actionable colors are used that include an action (such as moving you forward in the flow). Elements that are not actionable should not be in these colors. This applies to the Button, Link Button, and Text Link components. Proper use of these colors are defined further in this document in their respective sections. There are five colors used for actionable colors.
